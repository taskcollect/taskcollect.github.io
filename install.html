Install

Binary installers

End users should either install TaskCollect from their system's package manager (if available), or use one of the following binary installers to install TaskCollect (v1.0.6):

Google Android

  • 64-bit ARM☑⭳
  • 32-bit ARM☑⭳
  • 64-bit x86☑⭳
  • 32-bit x86☑⭳

Apple iOS

  • 64-bit ARM☑⭳
  • 64-bit x86☑⭳

Microsoft Windows

  • 64-bit x86☑⭳
  • 32-bit x86☑⭳

Apple macOS

  • 64-bit ARM☑⭳
  • 64-bit x86☑⭳

Linux

  • 64-bit x86☑⭳
  • 32-bit x86☑⭳
  • 64-bit ARM☑⭳
  • 32-bit ARM☑⭳

If your platform is not listed above, your platform may not be supported, or you may need to build from source.
